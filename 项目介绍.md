# 🏢 享+社区小程序项目介绍

## 📋 项目概述

**享+社区 (Enjoy+)** 是一个基于微信小程序的智慧社区服务平台，为居民提供一站式的社区生活服务。项目采用现代化的技术架构，实现了房屋管理、在线报修、访客邀请等核心功能模块。

## 🎯 核心功能

### 🏠 房屋管理模块
- 房屋信息维护与管理
- 位置定位与地址选择
- 房屋详情查看与编辑
- 房屋列表展示与删除

### 🔧 在线报修服务
- 报修申请提交
- 报修进度跟踪
- 历史记录查询
- 维修状态管理

### 👥 访客管理
- 访客信息录入
- 访客通行证生成
- 访客记录查询
- 访客权限管理

### 📢 社区服务
- 社区公告展示
- 实时通知推送
- 重要信息发布
- 社区活动宣传

## 🛠️ 技术架构

### 前端技术栈
- **框架**: 微信小程序原生开发
- **UI组件库**: Vant Weapp 1.10.4
- **网络请求**: wechat-http + 自定义拦截器
- **工具库**: 自定义工具函数集

### 主要依赖
```json
{
  "@vant/weapp": "^1.10.4",
  "wechat-http": "^0.0.8",
  "wechat-validate": "^0.0.3"
}
```

## ✨ 技术特色

### 🔐 安全认证体系

- 自动化token刷新机制
- 完整的权限控制流程
- 安全的登录状态管理

- **分包加载**: 模块化分包，提升加载速度

### 环境要求
- Node.js 14+
- 微信开发者工具
- 微信小程序账号

### 安装步骤
1. 克隆项目代码
2. 安装依赖: `npm install`
3. 构建npm: 微信开发者工具中构建npm
4. 编译运行: 在开发者工具中预览
